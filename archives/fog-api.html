用户系统

相关错误码

错误码 描述
101 用户不存在
102 用户名或密码不匹配
103 用户被禁用
104 用户已存在
105 用户名或密码不合法
106 注册失败（未知原因）

查询UID

请求地址: /api/user/uid
提交方法: GET/POST
说明: 通过用户输入的账号查询唯一uid
参数列表

参数 说明
username 用户名（可能是邮箱/用户名/手机号等）

请求示例
curl -d 'username=AngryPowman' 'http://47.90.123.165:8000/api/user/uid'

返回示例

用户注册

请求地址: /api/user/sign_up
提交方法: GET/POST
说明: 注册用户
参数列表

参数 说明
username 注册用户名
pwd 明文注册密码

请求示例
curl -d 'username=AngryPowman&pwd=123456' 'http://47.90.123.165:8000/api/user/sign_up'

返回示例

登录

请求地址: /api/user/sign_in
提交方法: GET/POST
说明: 登录账号并获取服务端返回的access token
参数列表

参数 说明
uid 用户唯一ID
pwd md5(明文密码)

请求示例
curl -d 'uid=10000&pwd=e10adc3949ba59abbe56e057f20f883e' 'http://47.90.123.165:8000/api/user/sign_in'

返回示例
